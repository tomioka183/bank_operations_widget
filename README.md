# Проект "Анализ Финансовых Операций"

## Описание Проекта
Этот проект предназначен для обработки и анализа списка финансовых операций. Он предоставляет базовую функциональность для фильтрации операций по их статусу и сортировки по дате, что помогает в удобном управлении и просмотре данных.

## Особенности
•   **Фильтрация по статусу:** Позволяет легко отбирать операции с определённым статусом (например, "EXECUTED" - выполненные, "PENDING" - в ожидании).
•   **Сортировка по дате:** Предоставляет возможность сортировать операции в хронологическом порядке (от старых к новым по умолчанию) или в обратном порядке (от новых к старым).
